ArmaLife

An RPG mod for Arma 3. http://www.altisliferpg.com

About

    ArmaLife is a github community, created for developing and updating the 'ArmaLife RPG Mod', AKA Altis Life, and originally created by TAW_Tonic.

History

    Altis Life development started at 10/09/13, by Tonic. After many releases, the v4.0 arrived and after that any new release was done by Tonic.
    For solving the bugs/exploits, danielstuart14 forked it. Some changes were done, and then, the first release was made, v4.0.2. At the time, it was pull-requested to the Altis Life git.
    Some time later, v4.1 was release. This was the first non-Tonic release, as Tonic gave to Daniel the permission for pushing changes to Altis-Life repo. Because git user repos are very limited, a new git organization was created, named ArmaLife.
    Then, ArmaLife started to be the development environment for Altis-Life, for testing and pull-requesting.
    Today, ArmaLife is the responsible for the AltisLife forum and all the 'life mod' releases.

Releases
